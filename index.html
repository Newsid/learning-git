    Introduction

    In this project, I will represent all the Skeletal System of the Thorax. The Skeletal System of the Thorax is composed by the thoracic vertebrae (12 vertebrae), the ribcage (12 couple of ribs), the sternum and the manubrium.

    Note: Being this a specular design, for each object was designed only the left side, and then copied using the S Transfrorm with parameter "-1"

    Note 2: In the beginning I tried using solids, but given the excessive load time, I opted for using surfaces.

    Note 3: Since I had no real measures, all proportions were calculated by eye.

    Main Structure of a Vertebra

    To create the main structure of a vertebra, I created two bases (upper and lower) and I joined via BEZIER(S1), using intermediate curves for the recess.

    Back of a Vertebra

    To create the back structure of a Vertebra, I created a series of NUBS starting from the center. The NUBS represent the contours of the structure and were combined by using BEZIER(S1). Ends were created separately using BEZIER.

    A Vertebra

    Now we can see the entire vertebra, following the combination of the main structure and the back structure.

    Ribs

    For each Rib I used series of NUBS joined by BEZIER, starting from the same point of the corresponding vertebra. Each rib has obviously a different curvature and end point.

    In this image We can see the 10° vertebra and the 10° left and right ribs.

    Sternum and Manubrium

    Sternum and Manubrium are the two bones on the front of the chest.

    I used these bones as a reference for the curvature of the ribs, lacking the real measurements and proportions.

    As for the previous, I used series of NUBS for the curves and BEZIER to generate the surface.

    Thorax

    Here we see the final design, obtained by the union of all individual objects.
